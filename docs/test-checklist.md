## 1. 基本動作チェック

- [ ] HTTPS でページが開けるか（`https://msyymmt.github.io/band-rehearsal-app/`）
- [ ] マイク許可ダイアログが出るか → 許可後にエラーが出ないか
- [ ] スペクトラムがリアルタイムで動くか
- [ ] 画面が真っ白・フリーズしないか

**不具合があればスクショを撮る**

---

## 2. キャリブレーション

各楽器ごとに単独で音を出して 3 回ずつサンプリング。

| 楽器 | サンプリングできたか | 記録された帯域（画面の値） | 体感と合っているか |
|------|:---:|---|---|
| ベース | [ ] | Hz – Hz | |
| ギター | [ ] | Hz – Hz | |
| ドラム | [ ] | Hz – Hz | |
| ボーカル | [ ] | Hz – Hz | |

**確認ポイント：**
- 楽器ごとに異なる帯域が記録されるか
- 極端に広い/狭い帯域になっていないか
- サンプリング中にスペクトラムがその楽器の音域でピークを示すか

## 3. 帯域バランス診断（全員で演奏中）

全員で演奏しながら以下を確認。

| 項目 | 結果 |
|------|------|
| スコア（0–100） | |
| ステータス（OK / 要調整 / 警告） | |
| ボーカルマスク検知 | あり / なし |
| 推奨事項の内容 | |

**確認ポイント：**
- ボーカルが聞こえづらい状況で「マスク検知」が出るか
- ギターの音量を下げた時にスコアが上がるか
- 推奨事項の内容が実態と合っているか
